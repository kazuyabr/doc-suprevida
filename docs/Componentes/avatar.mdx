import { Avatar, AvatarIcon } from "../../src/components/Avatar";

## Como utilizar?

Para utilizar o componente é necessário realizar a importação.

```tsx
import { Avatar, AvatarIcon } from "../components/Avatar";
```

O arquivo por padrão exporta dois componentes, `Avatar` e `AvatarIcon`. Após realizar a importação adicione o componente no arquivo `.tsx`.

**Utilização básica**:

```tsx
<Avatar online={false}>BP</Avatar>
```

<Avatar online={false}>BP</Avatar>

<br />

```tsx
<AvatarIcon
  online={false}
  img="https://www.w3schools.com/howto/img_avatar.png"
/>
```

<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={false}
/>

<br />

Para utilizar `AvatarIcon` é **obrigatório** passar a propriedade `img` com a URL ou PATH da imagem.

## Propriedades

### Online:

A propriedade `online` cria uma marcação no componente informando o status do usuário. `online` é obrigatório.

Valores disponíveis: `true` ou `false`

```tsx
<Avatar online={true}>BP</Avatar>
```

<Avatar online={true}>BP</Avatar>
<br />

```tsx
<Avatar online={false}>BP</Avatar>
```

<Avatar online={false}>BP</Avatar>
<br />

```tsx
<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={true}
/>
```

<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={true}
/>
<br />

```tsx
<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={false}
/>
```

<br />

<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={false}
/>

### Size:

A propriedade `size` altera o tamanho do componente. `size` não é obrigatória, seu valor padrão sempre vai ser: `small`.

Valores disponíveis: `md` ou `lg`

```tsx
<Avatar size="md" online={true}>
  BP
</Avatar>
```

<Avatar online={true} size="md">
  BP
</Avatar>
<br />

```tsx
<Avatar size="lg" online={true}>
  BP
</Avatar>
```

<Avatar online={true} size="lg">
  BP
</Avatar>

### Img:

A propriedade `img` adiciona uma imagem no componente. Para alterar a imagem adicione a URL ou PATH na propriedade. `img` é obrigatório e deve ser utilizada apenas no componente `AvatarIcon`.

```tsx
<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={true}
  size="lg"
/>
```

<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={true}
  size="lg"
/>
