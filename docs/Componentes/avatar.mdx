import { Avatar, AvatarIcon } from "../../src/components/Avatar";

## Como utilizar?

Para utilizar o componente é necessário realizar a importação.

```js
import { Avatar, AvatarIcon } from "../components/Avatar";
```

O arquivo por padrão exporta dois componentes, `Avatar` e `AvatarIcon`. Após realizar a importação do componente adicione a tag em seu arquivo `.tsx`.

**Utilização básica**:

```js
<Avatar online={false}>BP</Avatar>
```

<Avatar online={false}>BP</Avatar>

<br />

```js
<AvatarIcon
  online={false}
  img="https://www.w3schools.com/howto/img_avatar.png"
/>
```

<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={false}
/>

<br />

Para utilizar `AvatarIcon` é **obrigatório** passar a propriedade `img` com a URL ou PATH da imagem.

## Propriedades

### Online:

A propriedade `online` cria uma marcação no componente informando o status do usuário. `online` é obrigatório.

Valores disponíveis: `online={true}` ou `online={false}`

```js
<Avatar online={true}>
  BP
</Avatar>

<Avatar online={false}>
  BP
</Avatar>
```

<div>
  <Avatar online={true}>BP</Avatar>
  <br />
  <Avatar online={false}>BP</Avatar>
</div>

<br />

```js
<AvatarIcon img="https://www.w3schools.com/howto/img_avatar.png" online={true}/>

<AvatarIcon img="https://www.w3schools.com/howto/img_avatar.png" online={false}/>
```

<div>
  <AvatarIcon
    img="https://www.w3schools.com/howto/img_avatar.png"
    online={true}
  />
  <br />
  <AvatarIcon
    img="https://www.w3schools.com/howto/img_avatar.png"
    online={false}
  />
</div>

### Size:

A propriedade `size` altera o tamanho do componente. `size` não é obrigatória, seu valor padrão sempre vai ser: `small`.

Valores disponíveis: `size="md"` ou `size="lg"`

```js
<Avatar
  size="md"
  online={true} >
  BP
</Avatar>

 <Avatar
  size="lg"
  online={true} >
  BP
</Avatar>
```

<div>
  <Avatar online={true} size="md">
    BP
  </Avatar>
  <br />
  <Avatar online={true} size="lg">
    BP
  </Avatar>
</div>

### Img:

A propriedade `img` adiciona uma imagem no componente. `img` é obrigatório e deve ser utilizada apenas no componente `AvatarIcon`.

```js
<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={true}
  size="lg"
/>
```

<AvatarIcon
  img="https://www.w3schools.com/howto/img_avatar.png"
  online={true}
  size="lg"
/>
