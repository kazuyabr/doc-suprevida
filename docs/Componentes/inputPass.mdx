## Como utilizar?

Para utilizar o componente é necessário realizar a importação.

```tsx
import InputPass from "../components/InputPass";
```

Após realizar a importação adicione o componente no arquivo `.tsx`.

**Utilização básica**:

```tsx
<InputPass></InputPass>
```

<div style={{ width: "15rem" }}>
  <InputPass></InputPass>
</div>

## Propriedades

**Obs:** Todas as propriedades padrões de um componente HTML `input type="password"` estão disponíveis,
por exemplo: `onClick`, `placeholder`, `name` etc..
