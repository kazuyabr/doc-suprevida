## Como utilizar?

Para utilizar o componente é necessário realizar a importação.

```tsx
import ButtonIcon from "../components/ButtonIcon";
```

Após realizar a importação adicione o componente no arquivo `.tsx`.

**Utilização básica**:

```tsx
<ButtonIcon />
```

<ButtonIcon />

## Propriedades

### Color:

A propriedade `color` altera o estilo do componente de acordo com o tema configurado. `color` é obrigatório.

Valores disponíveis: `primary` `secondary` `disabled` `secondaryDisable`

```tsx
<ButtonIcon color="primary"></ButtonIcon>
```

<ButtonIcon color="primary"></ButtonIcon>
<br />

```tsx
<ButtonIcon color="disabled"></ButtonIcon>
```

<ButtonIcon color="disabled"></ButtonIcon>
<br />

```tsx
<ButtonIcon color="secondary"></ButtonIcon>
```

<ButtonIcon color="secondary"></ButtonIcon>
<br />

```tsx
<ButtonIcon color="secondaryDisable"></ButtonIcon>
```

<ButtonIcon color="secondaryDisable"></ButtonIcon>

### Progress:

A propriedade `progress` retira o ícone do botão e coloca uma circular progress no lugar. `progress` não é obrigatório e seu valor padrão sempre vai ser: `false`.

Valores disponíveis: `true`

```tsx
<ButtonIcon progress="true"></ButtonIcon>
```

<ButtonIcon progress="true"></ButtonIcon>

### Width:

A propriedade `width` altera a largura do componente. `width` não é obrigatório, seu valor padrão sempre vai ser: `0`.

```tsx
<ButtonIcon width="4rem" />
```

<ButtonIcon width="4rem" />

### Height:

A propriedade `height` altera a altura do componente. `height` não é obrigatório, seu valor padrão sempre vai ser: `0`.

```tsx
<ButtonIcon height="4.5rem" />
```

<ButtonIcon height="4.5rem" />
