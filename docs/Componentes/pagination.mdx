import Pagination from "../../src/components/Pagination";

## Como utilizar?

Para utilizar o componente é necessário realizar a importação.

```tsx
import Pagination from "../components/Pagination";
```

Após realizar a importação adicione o componente no arquivo `.tsx`.

**Utilização básica**:

```tsx
<Pagination count={3}></Pagination>
```

<Pagination count={3}></Pagination>

## Propriedades

### Count:

A propriedade `count` define o número de elementos do componente. `count` é obrigatório.

```tsx
<Pagination count={4}></Pagination>
```

<Pagination count={4}></Pagination>
