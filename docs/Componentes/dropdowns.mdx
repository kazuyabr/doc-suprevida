## Como utilizar?

Para utilizar o componente é necessário realizar a importação.

```tsx
import Dropdown from "../components/DropDown";
```

Após realizar a importação adicione o componente no arquivo `.tsx`.

**Utilização básica**:

```tsx
<Dropdown></Dropdown>
```

<Dropdown></Dropdown>
<br />

**Obs:** Todas as propriedades padrões de um componente HTML `select` estão disponíveis,
por exemplo: `onClick`, `onChange`, `name` etc..
